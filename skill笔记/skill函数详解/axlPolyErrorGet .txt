         axlPolyErrorGet (

         ==> t_error/nil
  SYNOPSIS

         Error type                      String returned

         bad data                        "Data probleminside polygon operations."
         internal error in               "Polygon operation failedbecause of internal error."
         logical op data handling
         numerical problem               "Computational problemwhile doing polygon operations."

         None.
  RETURNS
