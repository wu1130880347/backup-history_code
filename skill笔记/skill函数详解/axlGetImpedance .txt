NAME
         axlGetImpedance - return min/maximpedance of an object
  FUNCTION
         axlGetImpedance(
                 o_dbid
         ) => (f_min f_max)/nil
  SYNOPSIS
         Returns minimum and maximumimpedance for given item.
         Item can be either cline, cline segment, net or xnet.
         Impedance is in ohms bydefault.
  NEEDS
         segment cline
  RETURNS
         impedance in current mksunits.
  SEE ALSO
         axlSegDelayAndZ0
