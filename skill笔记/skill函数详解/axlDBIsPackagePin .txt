某个Pin是否是package的一个pin
  NAME
         axlDBIsPackagePin - Verify if a dbid is a PackagePin
  FUNCTION
         axlDBIsPackagePin (
                 o_dbid
         )
         ==> t or nil
  SYNOPSIS
         A "package pin" is defined as a "pin" with componentclass = IO
  NEEDS  
         o_dbid          dbid of the element to be checked.
  RETURNS
         t               If is a package pin.
         nil             If is not a package pin.
