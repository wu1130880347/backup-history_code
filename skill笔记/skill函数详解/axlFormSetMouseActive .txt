  NAME
         axlFormSetMouseActive - Sets theMouseActive option on a form
  FUNCTION
         axlFormSetMouseActive(
             r_form
         )
         ==> t/nil
  SYNOPSIS
         Sets the option to dispatchthe MouseActive event on a form.
  NEEDS
         r_form          Handle for the form
  RETURNS
         t      Option was set
         nil    r_form does not reference a valid form
