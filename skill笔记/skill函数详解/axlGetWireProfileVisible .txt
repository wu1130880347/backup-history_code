获取配置文件可见性
   NAME
          axlGetWireProfileVisible
   FUNCTION
          axlGetWireProfileVisible(t_profile)
             ==> t / nil
   SYNOPSIS
          This function willretrieve the visibility status of
          a bond wire profile. If noprofile matching the name supplied
          is found in the database, nil isreturned.
   NEEDS
          t_profile -- Name of profile toretrieve color for.
   RETURNS
          t if wire profile exists and isvisible.
          nil if profile isinvisible or does not exist.
