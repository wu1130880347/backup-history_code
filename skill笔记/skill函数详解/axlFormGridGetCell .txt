返回格点单元调用数据
  NAME
         axlFormGridGetCell - returns data ofgiven grid cell
  FUNCTION
         axlFormGridGetCell(
                 r_form
                 t_field
                 r_cell
         ) -> r_cell/nil
  SYNOPSIS
         Returns grid cell data for given row and column. Allassociated
         data for a cell is returned.Note the cell value is always
         returned as a string exceptfor REAL and LONG data types which
         is returned in their nativeformat.
         If row or cell number of 0 is used then top or side heading
         data is returned (if present).
         
         HINT: for best performancereuse the cell ifaccessing multiple
                 cells.
  NEEDS
         r_form: standard formhandle
         t_field: standard fieldname
         r_cell: standard grid cellfrom axlFormGridNewCell()
  RETURNS
         Returns cell for success, nil for invalid form id, field label
                 or if cell doesn't existin grid.
  SEE ALSO
         axlFormGridNewCell
  EXAMPLE
         ; Return value of cell - (1,3)
         cell = axlFormGridNewCell()
         cell->row = 3
         cell->col = 1
         axlFormGridInsertRows(form, "grid" cell)
         printf("cell value = %L\n", cell)
