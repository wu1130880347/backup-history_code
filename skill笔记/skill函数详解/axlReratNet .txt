  NAME
         axlReratNet - rerat a net
  FUNCTION
         axlReratNet(
                 t_netName/o_dbid
         )
         ==> t/nil
  SYNOPSIS
          Rerats a net.
          Normally this is notrequired since allegro automatically updates ratsnesting
          as required.
  NEEDS
         t_old_name      --     the existing net name.
         o_dbid          --      alt is a dbid that is on a net
  RETURNS
         t               --      the net is successfully renamed.
         nil             --      fails.
  EXAMPLE
         axlReratNet("NET1")
