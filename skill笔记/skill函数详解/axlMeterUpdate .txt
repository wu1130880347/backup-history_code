  SYNOPSIS

            done and/or the info stringmay be updated.
     NEEDS

            t_infoString - updatetext for progress meter info text line.
                         Value isone of:

                           ""  - clear info stringfield.
                       newText -update field with new text.
            t_infoStr2  -  (optional) Text for secondline
  
   RETURNS
